#printf 
> stdio.h에서 제공해주는 c언어의 대표적인 문장 출력용 기능(함수)<br>
> 일반적으로는 printf("작성하고싶은 문장");을 작성해 사용

예시)
```c
int main()
{
  printf("Good day to study!");
return 0;
}
```
